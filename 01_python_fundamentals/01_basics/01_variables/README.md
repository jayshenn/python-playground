# 变量和数据类型

## 学习目标

- 理解 Python 基本数据类型
- 掌握变量的声明和使用
- 了解类型转换

## 主要内容

### 基本数据类型

1. **数值类型**
   - `int` - 整数
   - `float` - 浮点数
   - `complex` - 复数

2. **文本类型**
   - `str` - 字符串

3. **布尔类型**
   - `bool` - True/False

4. **容器类型**
   - `list` - 列表（可变）
   - `tuple` - 元组（不可变）
   - `dict` - 字典
   - `set` - 集合
