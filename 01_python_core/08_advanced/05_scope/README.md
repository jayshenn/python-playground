# 作用域 (Scope)

本章节介绍变量的可访问范围以及 Python 搜索变量的 LEGB 规则。

## 核心知识点

1. **什么是作用域**：代码中定义变量的区域，决定了变量的可见性和生命周期。
2. **LEGB 规则**：Python 搜索变量的顺序：
    - **L (Local)**：局部作用域（函数内部）。
    - **E (Enclosing)**：嵌套作用域（外层非全局函数内部）。
    - **G (Global)**：全局作用域（模块级别）。
    - **B (Built-in)**：内置作用域（Python 解释器预定义）。

## 文件列表

- `01_what_is_scope.py`: 介绍作用域的基本概念及全局/局部区别。
- `02_legb_rule.py`: 通过代码演示完整的 LEGB 搜索优先级。
