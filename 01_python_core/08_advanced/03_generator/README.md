# 生成器 (Generators)

本章节介绍生成器的概念及其在处理大规模数据时的优势。

## 核心知识点

1. **什么是生成器**：一种特殊的迭代器，它不会一次性生成所有数据，而是在需要时才生成（惰性求值）。
2. **创建方式**：
    - **生成器表达式**：类似列表推导式，但使用小括号 `()`。
    - **生成器函数**：包含 `yield` 关键字的函数。
3. **yield 关键字**：用于暂停函数执行并返回一个值，下次调用时从暂停处继续。
4. **send() 方法**：除了获取值，还可以向生成器内部发送数据。
