# 闭包 (Closures)

本章节介绍 Python 中的闭包特性，即能够捕获外部函数作用域变量的内部函数。

## 核心知识点

1. **什么是闭包**：满足以下三个条件的函数：
    - 必须有内嵌函数。
    - 内嵌函数必须引用外部函数中的变量。
    - 外部函数返回值必须是内嵌函数。
2. **闭包的作用**：保持变量状态、封装逻辑、实现简单的工厂模式或装饰器的基础。
3. **查看闭包**：使用 `__closure__` 属性查看捕获的变量。

## 文件列表

- `01_what_is_closure.py`: 演示闭包的定义和三个基本条件。
- `02_use_closure.py`: 演示闭包在实际场景（如计数器）中的应用。
- `03_view_closure.py`: 介绍如何通过内置属性查看闭包捕获的具体值。
