# 装饰器 (Decorators)

本章节深入介绍 Python 中极其强大的装饰器模式，它是基于闭包实现的一种函数增强机制。

## 核心知识点

1. **什么是装饰器**：在不改变原函数代码和调用方式的情况下，动态添加额外功能（如日志、权限验证、性能测试等）。
2. **语法糖**：使用 `@装饰器名` 简化调用。
3. **函数装饰器**：最常见的装饰器形式，用于装饰普通函数。
4. **多层装饰器**：一个函数可以同时被多个装饰器叠加。
5. **带参数的装饰器**：通过在装饰器外再嵌套一层函数来接收配置参数。
6. **类装饰器**：通过实现 `__call__` 方法将类作为装饰器使用。
