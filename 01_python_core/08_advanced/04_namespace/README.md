# 命名空间 (Namespaces)

本章节介绍 Python 如何管理名称与对象之间的映射关系。

## 核心知识点

1. **什么是命名空间**：一个简单的映射（通常是字典），其中键是变量名，值是变量对应的对象。
2. **三种命名空间**：
    - **内置命名空间 (Built-in)**：Python 解释器启动时创建，包含内置函数和异常。
    - **全局命名空间 (Global)**：模块被加载时创建，包含模块级别的变量和函数。
    - **局部命名空间 (Local)**：函数调用时创建，函数返回时销毁。

- `02_three_namespaces.py`: 演示内置、全局和局部命名空间的共存与区别。
