# 属性 (Attributes)

本章节介绍 Python 中的类属性和实例属性的区别、用法及应用场景。

## 核心知识点

1. **类属性**：定义在类中、方法外的属性。由所有实例共享，可以通过类名或实例访问。
2. **实例属性**：通常在 `__init__` 方法中定义的属性。每个实例独有，互不干扰。
3. **优先级**：当实例属性与类属性同名时，实例对象访问的是实例属性（覆盖现象）。

## 文件列表

- `01_class_attribute.py`: 介绍类属性的定义和共享特性。
- `02_instance_attribute.py`: 介绍实例属性的定义和独立特性。
- `03_attribute_comparison.py`: 通过对比示例演示两者的相互影响和查找顺序。
