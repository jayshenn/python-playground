# 类的操作 (Class Operations)

本章节介绍如何对已定义的类进行操作，包括创建实例（对象）、访问属性、调用方法以及删除对象。

## 核心知识点

1. **实例化**：通过 `类名()` 的形式创建对象。
2. **属性访问**：使用点号运算符 `.` 访问或修改对象的属性。
3. **方法调用**：使用点号运算符 `.` 调用对象的方法。
4. **对象生命周期**：理解如何手动删除对象引用（`del` 关键字）。

## 文件列表

- `01_create_instance.py`: 演示如何实例化一个类。
- `02_access_attribute.py`: 演示如何访问和修改对象的属性。
- `03_call_method.py`: 演示如何调用类中定义的方法。
- `04_delete_object.py`: 演示如何删除对象引用。
