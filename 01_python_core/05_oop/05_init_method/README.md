# __init__() 方法 (Constructor Method)

本章节深入探讨 Python 类中的特殊方法 `__init__()`，即构造方法（或初始化方法）。

## 核心知识点

1. **构造方法的作用**：在创建对象时自动调用，用于初始化对象的属性。
2. **参数传递**：如何通过 `__init__` 方法接收外部传入的数据。
3. **注意事项**：`__init__` 不能有返回值（除了显式返回 `None`），且其第一个参数必须是 `self`。
