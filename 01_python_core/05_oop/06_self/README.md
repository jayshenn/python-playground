# self 参数 (The self Parameter)

本章节介绍 Python 类方法中的第一个参数 `self` 的含义及用法。

## 核心知识点

1. **self 的本质**：`self` 代表类的实例对象本身。它不是 Python 的关键字，但作为一种约定俗成的命名，必须保留。
2. **实例绑定**：通过 `self` 可以访问当前对象的属性和方法。
3. **方法间调用**：在类内部的一个方法中，通过 `self` 调用另一个方法。
