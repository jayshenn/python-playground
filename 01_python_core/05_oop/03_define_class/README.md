# 定义类 (Defining Classes)

本章节介绍如何在 Python 中定义一个类，包括基本语法、命名规范以及文档字符串的使用。

## 核心知识点

1. **基本语法**：使用 `class` 关键字。
2. **命名规范**：遵循 PEP 8，使用大驼峰命名法（PascalCase）。
3. **类体结构**：包含属性（变量）和方法（函数）。
4. **文档字符串**：为类提供说明，增强代码可读性。

## 文件列表

- `01_basic_syntax.py`: 介绍类的基本定义语法。
- `02_naming_convention.py`: 展示类的命名规范（大驼峰）。
- `03_simple_class.py`: 一个包含基本属性和方法的简单类示例。
- `04_docstring.py`: 演示如何为类编写和访问文档字符串。
