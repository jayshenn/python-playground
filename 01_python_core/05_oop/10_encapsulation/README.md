# 封装 (Encapsulation)

本章节介绍面向对象编程中的封装特性，即如何隐藏对象的内部细节并提供安全的操作接口。

## 核心知识点

1. **私有化**：在属性或方法名前加上双下划线 `__`，使其在类外部无法直接访问。
2. **私有属性与方法**：用于保护核心数据和内部逻辑，防止被外部非法修改。
3. **Getter/Setter**：通过自定义方法或 `@property` 装饰器，实现对私有属性的受控访问。
4. **property 装饰器**：将类的方法伪装成属性，使调用更加直观且符合 Pythonic 风格。

## 文件列表

- `01_privatization.py`: 介绍 Python 中私有化的概念和实现方式。
- `02_private_attribute.py`: 演示如何定义和使用私有属性。
- `03_private_method.py`: 演示如何定义和使用私有方法。
- `04_property.py`: 演示 `@property` 装饰器的用法，实现属性的拦截和校验。
