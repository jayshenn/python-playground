# 方法 (Methods)

本章节介绍 Python 类中定义的各种方法类型，包括实例方法、类方法、静态方法以及特殊方法（魔法方法）。

## 核心知识点

1. **实例方法**：最常用的方法，第一个参数是 `self`，用于操作实例属性。
2. **类方法**：使用 `@classmethod` 装饰，第一个参数是 `cls`，用于操作类属性。
3. **静态方法**：使用 `@staticmethod` 装饰，没有默认的 `self` 或 `cls` 参数，逻辑上属于类但不需要访问类或实例的状态。
4. **特殊方法**：以双下划线开头和结尾的方法（如 `__str__`），由 Python 解释器在特定时刻自动调用。
