# 端口 (Ports)

本章节介绍网络通信中的逻辑接口——端口。

## 核心知识点

1. **什么是端口**：设备与外界通讯交流的出口，用于区分同一台主机上的不同网络服务。
2. **端口号范围**：0 - 65535。
3. **端口分类**：
    - **知名端口 (Well-known Ports)**: 0 - 1023，分配给特定的核心服务（如 HTTP 80, HTTPS 443）。
    - **注册端口 (Registered Ports)**: 1024 - 49151，分配给用户进程或应用（如 MySQL 3306, Redis 6379）。
    - **动态/私有端口 (Dynamic Ports)**: 49152 - 65535，由系统动态分配。
