# 套接字 (Socket)

本章节介绍网络通信的基石——Socket。

## 核心知识点

1. **什么是 Socket**：应用层与传输层之间的软件抽象层，它是一组接口。
2. **Socket 作用**：使程序员能方便地进行网络通信，而不需要深入了解底层协议细节。
3. **Python socket 模块**：提供了丰富的 API 来创建和操作 Socket。

## 文件列表

- `01_what_is_socket.py`: 介绍 Socket 的定义和基本工作方式。
- `02_socket_usage.py`: 演示在 Python 中创建和初始化 Socket 的基本步骤。
