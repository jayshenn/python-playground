# 网络基础 (Network Basics)

本章节介绍进行网络编程前必须了解的核心概念。

## 核心知识点

1. **网络编程三要素**：
    - **IP 地址**：用于标识网络中的主机。
    - **端口号 (Port)**：用于标识主机上的特定应用程序。
    - **通信协议**：数据传输的规则（如 TCP, UDP, HTTP）。
2. **TCP/IP 协议族**：现代互联网的基石，采用分层模型（应用层、传输层、网络层、链路层）。
