# IP 地址 (IP Addressing)

本章节深入介绍 IP 地址的原理、分类及现代网络中的应用。

## 核心知识点

1. **IP 定义**：互联网协议地址，分为 IPv4 和 IPv6。
2. **子网掩码**：用于区分 IP 地址中的网络号和主机号。
3. **地址分类**：A、B、C 类地址及其范围。
4. **公网与私网**：理解 NAT 技术及私有地址段（如 192.168.x.x）。
5. **IPv6**：解决 IPv4 地址枯竭问题的下一代互联网协议。

## 文件列表

- `01_what_is_ip.py`: 介绍 IP 地址的基本概念及查询。
- `02_subnet_mask.py`: 解释子网掩码的原理。
- `03_ipv4_classification.py`: 展示 A/B/C 三类地址的范围。
- `04_public_private.py`: 介绍公网与私网地址的区别。
- `05_ipv4_vs_ipv6.py`: 对比 IPv4 与 IPv6 的优劣。
