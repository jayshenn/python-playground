# HTTP (HyperText Transfer Protocol)

本章节介绍互联网应用最广泛的应用层协议——HTTP。

## 核心知识点

1. **什么是 HTTP**：超文本传输协议，基于 TCP 协议，遵循“请求-响应”模式。
2. **报文结构**：
    - **请求报文**：请求行、请求头、空行、请求体。
    - **响应报文**：状态行、响应头、空行、响应体。
3. **请求方法**：GET (获取)、POST (提交)、PUT (更新)、DELETE (删除) 等。
4. **状态码**：200 (成功)、404 (未找到)、500 (服务器错误) 等。
