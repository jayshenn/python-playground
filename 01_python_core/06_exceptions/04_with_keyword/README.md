# with 关键字 (The with Statement)

本章节介绍 Python 的上下文管理器（Context Manager），即 `with` 关键字的用法及其背后的工作原理。

## 核心知识点

1. **基本语法**：使用 `with` 语句简化资源管理代码（如文件操作）。
2. **上下文管理协议**：理解 `__enter__` 和 `__exit__` 魔法方法。
3. **优势**：确保即使在发生异常时，资源（如文件、连接）也能被正确关闭，避免资源泄露。

## 文件列表

- `01_syntax.py`: 介绍 `with` 语句的基本用法。
- `02_working_principle.py`: 通过自定义类演示上下文管理器的工作流程。
- `03_file_case.py`: 实战演练：验证 `with` 在文件处理异常时的安全性。
