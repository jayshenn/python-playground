# 异常处理 (Exception Handling)

本章节介绍 Python 中处理异常的完整语法结构。

## 核心知识点

1. **try-except**：尝试执行代码，如果出错则捕获并处理。
2. **多重捕获**：针对不同类型的异常提供不同的处理方案。
3. **获取异常信息**：使用 `as e` 获取异常的具体描述。
4. **else 块**：当 `try` 块中没有发生任何异常时执行的代码。
5. **finally 块**：无论是否发生异常，都必须执行的代码（通常用于资源释放）。

## 文件列表

- `01_try_except.py`: 介绍最基础的异常捕获语法。
- `02_catch_specific.py`: 演示如何捕获特定异常并获取错误信息。
- `03_else.py`: 介绍 `else` 块在异常处理中的作用。
- `04_finally.py`: 介绍 `finally` 块的使用场景。
