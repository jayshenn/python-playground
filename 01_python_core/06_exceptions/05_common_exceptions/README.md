# 常见异常 (Common Exceptions)

本章节梳理 Python 标准库中常见的异常类型及其层级关系。

## 核心知识点

1. **异常层级**：理解 `BaseException`, `Exception`, `ArithmeticError` 等基类的关系。
2. **常见具体异常**：熟悉 `ValueError`, `TypeError`, `KeyError`, `IndexError` 等异常的触发场景。
3. **捕获策略**：根据层级关系，决定是捕获具体异常还是通用异常。

## 文件列表

- `01_exception_hierarchy.py`: 展示 Python 异常的类继承结构。
- `02_specific_exceptions.py`: 列举并演示最常用的具体异常类型。
