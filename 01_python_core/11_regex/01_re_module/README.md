# re 模块 (The re Module)

本章节介绍 Python 中处理正则表达式的标准库 `re`。

## 核心知识点

1. **基本匹配**：
    - `re.match()`: 从字符串开头匹配。
    - `re.search()`: 在整个字符串中搜索第一个匹配项。
2. **全局搜索**：
    - `re.findall()`: 返回所有匹配项的列表。
    - `re.finditer()`: 返回所有匹配项的迭代器。
3. **文本处理**：
    - `re.sub()`: 替换匹配到的文本。
    - `re.split()`: 根据正则模式分割字符串。

## 文件列表

- `01_search.py`: 演示如何在文本中查找匹配内容。
- `02_match.py`: 演示从头匹配的用法。
- `03_findall.py`: 演示全局匹配的用法。
- `04_sub.py`: 演示文本替换的用法。
- `05_split.py`: 演示文本分割的用法。
