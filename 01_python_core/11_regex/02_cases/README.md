# 正则表达式案例 (Regex Practice Cases)

本章节通过一系列常用的小案例，演示正则表达式在实际开发中的应用。

## 核心案例

1. **号码验证**：匹配合法的中国手机号码。
2. **邮箱验证**：匹配常见的电子邮箱格式。
3. **数字提取**：从混合文本中精确提取数值信息。
4. **URL 提取**：从文本中识别并提取超链接。
5. **复杂替换**：使用 `re.sub` 结合回调函数进行复杂的文本处理。

## 文件列表

- `01_match_phone.py`: 实战：匹配手机号码。
- `02_match_email.py`: 实战：匹配电子邮箱。
- `03_extract_numbers.py`: 实战：从混合文本中提取数字。
- `04_extract_url.py`: 实战：获取文本中的网址。
- `05_replace_numbers.py`: 实战：将数字替换为文字描述。
