# 泛型编程 (Generics)

本章节介绍了 Python 中的泛型编程，这是构建可重用、类型安全的库和框架的核心工具。

## 目录

- [01_type_var.py - TypeVar 类型变量](01_type_var.py)：介绍如何定义类型变量、约束变量以及指定上界。
- [02_generic_class.py - 泛型类](02_generic_class.py)：介绍如何创建处理多种类型的泛型容器和业务类。
- [03_type_var_tuple.py - TypeVarTuple](03_type_var_tuple.py)：介绍如何处理可变数量的类型参数。
- [04_param_spec.py - ParamSpec](04_param_spec.py)：介绍如何捕获并保留函数的参数签名。

## 核心概念

1. **TypeVar**：泛型的基石，代表一个位置占位符类型。
2. **泛型函数/类**：在定义时不指定具体类型，而在使用时（通过推断或显式指定）确定类型。
3. **ParamSpec**：专门用于装饰器的类型变量，解决了装饰器丢失原函数参数信息的问题。
4. **TypeVarTuple**：支持形状未定的元组或数组的类型注解。

## 学习建议

- Python 3.12+ 引入了更简洁的 `[T]` 语法，本章节将重点展示这一新语法。
- 只有在需要兼容旧版本或需要特定高级特性（如协变/逆变）时，才建议使用传统的 `TypeVar` 定义方式。
