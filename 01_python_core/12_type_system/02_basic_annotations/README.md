# 基础类型注解 (Basic Type Annotations)

本章节介绍如何对 Python 的内置类型、集合类型以及函数进行基础的类型标注。

## 核心知识点

1. **内置类型**：直接使用 `int`, `str`, `float`, `bool` 等。
2. **集合类型**：使用 `list`, `dict`, `set`, `tuple`（Python 3.9+ 支持直接使用小写内置类）。
3. **类型别名**：为复杂的类型定义一个简短的名字，提高复用性。
4. **函数注解**：标注参数类型及返回值类型。

## 文件列表

- `01_builtin_types.py`: 演示标量类型的注解。
- `02_collection_types.py`: 演示容器类型的注解。
- `03_type_alias.py`: 演示如何创建类型别名。
- `04_function_annotations.py`: 演示函数的完整注解方案。
