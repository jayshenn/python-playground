# 文件拷贝案例

对应文档：`05-file-operations.md` § 5.8

## 简介
本目录包含多种文件拷贝场景的实现代码，从基础到高级。

## 文件列表

- `01_basic_text_copy.py`: 基本的文本文件拷贝。
- `02_binary_copy.py`: 二进制文件拷贝。
- `03_chunk_copy.py`: 分块拷贝大文件，节省内存。
- `04_copy_with_progress.py`: 带进度显示的拷贝实现。
- `05_shutil_copy.py`: 使用 `shutil` 模块进行简便拷贝。
- `06_batch_copy.py`: 批量拷贝指定类型的文件。
- `07_complete_copy_tool.py`: 一个功能较完整的文件拷贝工具函数。
