# 最佳实践

对应文档：`05-file-operations.md` § 5.9

## 简介
本目录总结了 Python 文件操作中的最佳实践。

## 文件列表

- `01_with_statement.py`: 演示为什么要始终使用 `with` 语句。
- `02_specify_encoding.py`: 演示为什么要显式指定文件编码。
- `03_handle_exceptions.py`: 演示正确的文件异常处理方式。
- `04_pathlib_usage.py`: 演示使用 `pathlib` 替代传统路径操作。
- `05_process_large_files.py`: 演示如何高效处理大文件。
