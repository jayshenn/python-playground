# 文件打开与关闭

对应文档：`05-file-operations.md` § 5.4

## 简介
本目录包含关于 Python 文件打开和关闭操作的示例代码，包括 `open()` 函数的使用、不同的文件模式以及如何正确关闭文件。

## 文件列表

- `01_open_function.py`: 使用 `open()` 函数打开文件的基本语法。
- `02_file_modes.py`: 演示不同的文件打开模式（`r`, `w`, `a`, `r+`, `rb`, `wb` 等）。
- `03_close_file.py`: 演示手动关闭文件和使用 `with` 语句自动关闭文件。
- `04_file_path.py`: 演示相对路径、绝对路径以及使用 `pathlib` 处理路径。
