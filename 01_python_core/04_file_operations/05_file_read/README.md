# 文件读操作

对应文档：`05-file-operations.md` § 5.6

## 简介
本目录包含关于 Python 文件读取操作的示例代码。

## 文件列表

- `01_read_method.py`: 演示 `read()` 方法，包括读取全部和指定字节数。
- `02_readline_method.py`: 演示 `readline()` 方法，读取单行。
- `03_readlines_method.py`: 演示 `readlines()` 方法，读取所有行到列表。
- `04_iterate_file.py`: 演示直接迭代文件对象（推荐方式）。
- `05_read_methods_comparison.py`: 对比不同读取方法的性能和适用场景。
- `06_file_pointer.py`: 演示文件指针操作 `tell()` 和 `seek()`。
