# 文件写操作

对应文档：`05-file-operations.md` § 5.5

## 简介
本目录包含关于 Python 文件写入操作的示例代码。

## 文件列表

- `01_write_method.py`: 演示 `write()` 方法的使用。
- `02_writelines_method.py`: 演示 `writelines()` 方法的使用。
- `03_write_data_types.py`: 演示如何写入字符串、数字、JSON 和二进制数据。
- `04_write_modes_comparison.py`: 对比 `w` (覆盖), `a` (追加), `x` (独占创建) 模式。
- `05_buffer_flush.py`: 演示文件缓冲机制和 `flush()` 方法的使用。
