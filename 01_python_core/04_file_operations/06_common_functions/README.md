# 常用函数和方法

对应文档：`05-file-operations.md` § 5.7

## 简介
本目录介绍 Python 中处理文件和目录的常用模块和函数。

## 文件列表

- `01_file_methods.py`: 汇总演示文件对象的常用属性和方法。
- `02_os_module.py`: 演示 `os` 模块进行文件和目录操作。
- `03_shutil_module.py`: 演示 `shutil` 模块进行高级文件操作（复制、移动、归档）。
- `04_pathlib_module.py`: 演示 `pathlib` 模块（面向对象路径处理，推荐）。
- `05_glob_module.py`: 演示 `glob` 模块进行文件模式匹配查找。
