# 进程与线程对比 (Process vs Thread Comparison)

本章节通过横向对比，帮助开发者在实际项目中选择最合适的并发模型。

## 核心知识点

1. **资源占用**：进程独立内存，开销大；线程共享内存，开销小。
2. **通信方式**：进程间需要 IPC (Queue, Pipe)；线程间可以直接访问全局变量。
3. **安全性**：进程崩溃不影响其他进程；一个线程崩溃可能导致整个进程崩溃。
4. **性能限制**：多进程可利用多核（并行）；多线程受 GIL 限制（并发）。
